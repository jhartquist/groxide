---

Add [profile.release] section to Cargo.toml with:
- lto = "thin" (link-time optimization for smaller/faster binaries)
- codegen-units = 1 (better optimization at cost of slower release builds)
- strip = true (strip debug symbols from release binaries)

These matter for a CLI tool that users install via cargo install — smaller and faster binaries.
