title: "Add Rust build caching to CI"
model: Opus
completed: true
---

Add Swatinem/rust-cache action to all CI jobs in .github/workflows/ci.yml. This caches the cargo registry, target directory, and incremental compilation artifacts. Builds will go from minutes to seconds on cache hits.

Add the cache step after the toolchain setup step in each job (check, fmt, clippy, test).
