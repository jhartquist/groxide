Add a security scanning job to .github/workflows/ci.yml that runs:
1. cargo-audit (via rustsec/audit-check action or cargo install + run) to check for known vulnerabilities in dependencies
2. cargo-deny (via EmbarkStudios/cargo-deny-action) to enforce dependency policies

Create a deny.toml config file with sensible defaults:
- Ban duplicate versions of key crates
- Deny known-bad licenses
- Check for unmaintained crates
- Check for yanked crates
