## Move dependency crate cache from target/groxide/ to ~/.cache/groxide/deps/

**Plan reference:** docs/plans/2026-03-05-cli-redesign-implementation.md — Phase 3, Task 11

Dependencies are versioned and immutable (like external crates). Two projects using serde@1.0.210 should share the same cached index. Move dependency caching to the global cache directory.

### What to do

1. **Update `cache_path` for `CrateSource::Dependency`** in `src/cache.rs:58-68`:

```rust
CrateSource::Dependency { name, version, .. } => {
    let cache_dir = dirs::cache_dir()?;
    let filename = format!("{name}-{version}{feature_suffix}.groxide");
    Some(cache_dir.join("groxide").join("deps").join(filename))
}
```

2. **Update tests:**
   - Change `cache_path_dependency_under_target` to `cache_path_dependency_uses_global_cache`
   - Verify path contains `groxide/deps/` instead of `target/groxide/`
   - Update `round_trip_dependency_save_load` if it assumes target dir

3. **Update `clear_global_cache`** (from Task 52) to also clear the `deps/` subdirectory (it already does since it removes the whole `groxide/` dir).
