Move clippy and rustc lint configuration into Cargo.toml [lints] section instead of only passing flags on the command line. This ensures clippy catches things in IDEs and CI without needing the exact right flags.

Add [lints.clippy] and [lints.rust] sections with the pedantic warnings currently passed via CLI flags (clippy::pedantic as warn, warnings as deny). Update mise.toml and CI to rely on the Cargo.toml config rather than inline flags.

Reference: current flags are `cargo clippy --all-targets -- -W clippy::pedantic -D warnings` in both mise.toml and .github/workflows/ci.yml.
