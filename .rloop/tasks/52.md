## Add --clear-cache flag to wipe global documentation cache

**Plan reference:** docs/plans/2026-03-05-cli-redesign-implementation.md — Phase 1, Task 4

Add a flag that wipes ~/.cache/groxide/ and exits. One flag, no arguments, clears everything. Indexes rebuild automatically on next query.

### What to do

1. **Add field to `Cli` struct** in `src/cli.rs`:

```rust
/// Clear the global documentation cache and exit
#[arg(long)]
pub clear_cache: bool,
```

2. **Add `clear_global_cache` function** to `src/cache.rs`:

```rust
/// Removes the global cache directory.
///
/// Returns the path that was cleared, or `None` if the cache dir could not be determined.
pub(crate) fn clear_global_cache() -> Option<PathBuf> {
    let cache_dir = dirs::cache_dir()?.join("groxide");
    if cache_dir.exists() {
        let _ = fs::remove_dir_all(&cache_dir);
    }
    Some(cache_dir)
}
```

3. **Handle at the top of `run()` in `src/lib.rs`**, before any project discovery:

```rust
if cli.clear_cache {
    if let Some(path) = cache::clear_global_cache() {
        eprintln!("[grox] Cleared cache at {}", path.display());
    } else {
        eprintln!("[grox] No cache directory found");
    }
    return Ok(());
}
```

4. **Write integration test** in `tests/cli_happy.rs`:

```rust
#[test]
fn clear_cache_exits_successfully() {
    let output = Command::new(cargo_bin!("grox"))
        .arg("--clear-cache")
        .output()
        .expect("command runs");
    assert!(output.status.success());
    let stderr = String::from_utf8_lossy(&output.stderr);
    assert!(stderr.contains("[grox]"));
}
```

5. **Write unit test** for `clear_global_cache` in `src/cache.rs` tests.
