Add matrix builds to the test job in .github/workflows/ci.yml to test on both ubuntu-latest and macos-latest.

groxide shells out to cargo and rustdoc, handles crates.io downloads, and manages file paths — all things that can behave differently across platforms. The check/fmt/clippy jobs can stay ubuntu-only (platform-independent), but the test job should run on both.

Use strategy.matrix.os: [ubuntu-latest, macos-latest] on the test job.
