Add a coverage job to .github/workflows/ci.yml using cargo-llvm-cov and codecov:

1. Add a new 'coverage' job that:
   - Installs cargo-llvm-cov (via taiki-e/install-action)
   - Runs cargo llvm-cov --lcov --output-path lcov.info
   - Uploads to Codecov via codecov/codecov-action@v4
2. Add a codecov badge to README.md

This gives visibility into test coverage gaps without being prescriptive about a minimum threshold.
