Final quality sweep before open source release.

## Checklist

1. `cargo fmt -- --check` — formatting clean
2. `cargo clippy --all-targets` — zero warnings (pedantic enabled in Cargo.toml)
3. `cargo test` — all tests pass
4. `INSTA_UPDATE=always cargo test` — snapshots up to date
5. No remaining `// TODO`, `// HACK`, `// FIXME`, `// XXX` comments
6. No `.unwrap()` in library code (test code is fine)
7. All `pub(crate)` boundaries are intentional
8. Doc comments (`///`) on all public and pub(crate) functions
9. No file exceeds ~800 lines
10. No function exceeds ~50 lines
11. No duplicated constants or near-identical function pairs
12. No hacks (serialize roundtrip, blank-line counting, etc.)
13. Module structure has clear single responsibilities
14. Error messages are helpful and consistent
15. All `expect()` calls have `"invariant: ..."` messages explaining the guarantee

## Action

If any items fail, fix them. This is a cleanup task, not just a check.
