Add a new CI job to `.github/workflows/ci.yml` that verifies the declared MSRV is satisfiable.

Add a job called `msrv` that:
1. Checks out the code
2. Installs the MSRV toolchain using `dtolnay/rust-toolchain` with `toolchain: "1.85"` (read the current rust-version from Cargo.toml to stay in sync)
3. Uses `Swatinem/rust-cache@v2`
4. Runs `cargo check --all-targets`

Place it after the existing `check` job. This ensures the MSRV declaration stays honest as dependencies evolve.
